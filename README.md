 Project Overview:-This project explores the theoretical and practical aspects of Open Source Software, focusing on the Python ecosystem and Linux system management. It includes a series of automated shell scripts designed to audit system identity, package installations, and security permissions.
 Software Choice: PythonOrigin: Created by Guido van Rossum and released in 1991 to provide a powerful yet easy-to-use alternative to C/C++.License: Distributed under the PSF (Python Software Foundation) License, a permissive open-source license that allows users to run, modify, and distribute the code freely.Impact: Python has enabled massive growth in Artificial Intelligence (AI), Data Science, and Web Development through libraries like NumPy and TensorFlow.
 Included Shell:-
      1.Scripts1. System Identity Report (script1.sh)Displays the auditor's identity (Ashmi) and the chosen software (Python).Reports the current Linux Kernel version and system date.
      2. FOSS Package Inspector (script2.sh)Automates a check to see if python3 is installed on the local system.Provides a brief description of the package using a case statement logic.
      3. Disk & Permission Auditor (script3.sh)Audits critical system directories including ., /tmp, and /usr.Extracts and displays specific permission strings, owners, and groups to ensure security compliance.
      4. Log File Analyzer (script4.sh)Scans test.log for the keyword "error" and calculates the total number of occurrences.Outputs the most recent error matches to the terminal for quick debugging.
      5. Open Source Manifesto Generator (script5.sh)An interactive script that collects personal views on software freedom.Saves a final Open Source Manifesto to manifesto.txt, documenting a future goal to build AI projects.
